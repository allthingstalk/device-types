# ELSYS ERS Eco CO2 Lite LoRa

> Device Type: `elsys/ers-eco-co2-lite-lora`  

[Operating Manual](https://elsys.se/public/manuals/New/Operating-Manual-ERS-Eco-series-LoRa-1.pdf)  
[Datasheet](https://elsys.se/public/datasheets/ERS_Series_LoRa_Datasheet.pdf)


ERS Eco CO2 Lite LoRa is a LoRaWAN® temperature, humidity, and CO2 sensor for the indoor environment. This sensor is your environmentally friendly option, with an organic solar cell as the only power source and an enclosure made from biodegradable material.

Use the general Elsys JS payload decoder.  

## Assets
| Name        | Title           | Unit | Data Type | Asset Type |

| humidity    | Humidity        | %    | number    | sensor     |
| temperature | Temperature     | °C   | number    | sensor     |
| battery     | Battery Voltage | V    | number    | sensor     |
